# 装饰系统说明文档

## 📋 设计理念

**装饰为主 (70%) + 升级为辅 (30%)**

- ✅ **个性化表达** - 每个用户小屋都独特
- ✅ **长期可运营** - 可持续上新装饰，用户收集欲强
- ✅ **开发成本低** - 模块化资源，不需要多套房屋模型
- ✅ **商业化潜力** - 限定装饰/盲盒/包装，变现路径清晰

---

## 🏠 小屋等级与槽位

### 等级配置

| 等级 | 装饰槽位 | 所需经验 | 解锁内容 |
|------|----------|----------|----------|
| Lv.1 | 5 个 | 0 | 基础家具 |
| Lv.2 | 8 个 | 100 | + 墙面装饰 |
| Lv.3 | 12 个 | 300 | + 地面装饰 |
| Lv.4 | 16 个 | 600 | + 植物 |
| Lv.5 | 20 个 | 1000 | + 特效/宠物 |
| Lv.6 | 25 个 | 1500 | - |
| Lv.7 | 30 个 | 2200 | - |
| Lv.8 | 35 个 | 3000 | - |
| Lv.9 | 40 个 | 4000 | - |
| Lv.10 | 50 个 | 5000 | 满级 |

### 升级方式

- **装饰获得经验** - 装备装饰时，温暖度 × 10 = 经验值
- **升级奖励** - 解锁更多装饰槽位

## 🎨 装饰物品系统

### 槽位类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `FURNITURE` | 家具 | 桌子、沙发、书架 |
| `WALL` | 墙面装饰 | 挂画、壁纸、窗户 |
| `FLOOR` | 地面装饰 | 地毯、地板 |
| `PLANT` | 植物 | 盆栽、花卉 |
| `EFFECT` | 特效 | 音乐盒、光效、宠物 |

### 稀有度

| 稀有度 | 颜色 | 说明 |
|--------|------|------|
| 1 - 普通 | ⚪ 白色 | 基础装饰，价格低 |
| 2 - 稀有 | 🔵 蓝色 | 较少见，中等价格 |
| 3 - 史诗 | 🟣 紫色 | 罕见，高价格 |
| 4 - 传说 | 🟡 金色 | 限定，极高价格 |

### 温暖度

- 每件装饰有温暖度加成
- 装备装饰 → 增加小屋温暖度
- 卸下装饰 → 减少温暖度
- 温暖度影响排行榜排名

## 📖 装饰图鉴系统

### 装饰系列

每个系列包含若干装饰物品，完成收集可获得奖励。

**示例系列**:

#### 1. 温馨家具系列
- 包含：木质桌子、舒适沙发、小台灯、书架、温馨地毯、艺术挂画
- 完成奖励：🦴100 + 🐟10

#### 2. 春日限定系列
- 包含：绿色植物、鲜花瓶、樱花盆栽、春天地毯
- 完成奖励：🐟20 + 💎5

#### 3. 梦幻特效系列
- 包含：音乐盒、水晶吊灯、星空壁纸
- 完成奖励：💎10

### 收集进度

- 自动追踪用户拥有的装饰
- 显示收集百分比
- 完成时自动发放奖励
- 发放通知提醒

## 🏆 温暖度排行榜

### 排行规则

- 按小屋温暖度降序排列
- 显示前 50 名用户
- 实时更新

### 展示信息

- 排名（🥇🥈🥉）
- 用户昵称 + 头像
- 小屋等级
- 温暖度数值
- 小屋主题

### 统计信息

- 参与人数
- 最高温暖度
- 平均温暖度
- 平均小屋等级

## 🔧 API 使用示例

### 获取我的小屋

```bash
GET /api/cottage/my
```

响应：
```json
{
  "cottage": {
    "id": 1,
    "level": 3,
    "experience": 250,
    "warmth": 45,
    "maxSlots": 12,
    "usedSlots": 8,
    "nextLevel": {
      "level": 4,
      "expNeeded": 600,
      "maxSlots": 16
    }
  },
  "decorations": [...],
  "levelConfig": {...}
}
```

### 装备装饰

```bash
POST /api/cottage/decorate
{
  "userDecorationId": 5,
  "slotPosition": 3,
  "isEquipped": true
}
```

### 获取图鉴列表

```bash
GET /api/cottage/collections
```

### 获取温暖度排行榜

```bash
GET /api/cottage/warmth-ranking?limit=50&offset=0
```
