# 📸 设备绑定功能 - 截图指南

## 快速开始

### 方法一：手动截图（推荐，最快）

#### 1. 打开 HTML 原型
在浏览器中打开：
```
/root/.openclaw/workspace/device_binding_demo.html
```

或直接双击文件。

#### 2. 开启开发者模式
- **Chrome/Edge**: 按 `F12` 或 `Ctrl+Shift+I`
- **Firefox**: 按 `F12`
- **Safari**: `开发` → `进入响应式设计模式`

#### 3. 选择移动设备视图
点击开发者工具中的 **设备切换图标** (📱)，选择：
- iPhone 12 Pro (390x844) 或
- iPhone X (375x812) 或
- Pixel 5 (393x851)

#### 4. 按步骤截图

| 序号 | 文件名 | 操作 | 截图内容 |
|------|--------|------|----------|
| 1 | `step1_select_device.png` | 打开页面 | 选择设备类型页面（两个卡片） |
| 2 | `step2_guide_glucose.png` | 点击血糖仪卡片 | 血糖仪操作指引 |
| 3 | `step2_guide_bp.png` | 返回后点击血压计卡片 | 血压计操作指引 |
| 4 | `step3_searching.png` | 点击"已阅读，开始绑定" | 搜索动画页面 |
| 5 | `step4_device_list.png` | 等待搜索完成（或修改 HTML 模拟） | 设备列表页面 |
| 6 | `step5_pairing.png` | 点击设备 | 配对确认页面 |
| 7 | `step6_success.png` | 点击确认配对 | 绑定成功页面 |

#### 5. 保存截图
- **Chrome**: 右键 → 截图 → 捕获节点截图 / 捕获全屏截图
- **或使用系统截图工具**: `Win+Shift+S` / `Cmd+Shift+4`

---

### 方法二：使用 Python 脚本生成示意图

运行脚本自动生成带标注的界面示意图：

```bash
cd /root/.openclaw/workspace
/tmp/docx_env/bin/python generate_screenshots.py
```

生成的图片会保存到 `screenshots/` 目录。

## 截图后：插入到 Word 文档

### 步骤
1. 打开 `设备绑定功能需求文档.docx`
2. 找到占位符：`【此处插入截图：xxx.png】`
3. 删除占位符文字
4. 点击 `插入` → `图片` → `此设备`
5. 选择对应的截图文件
6. 调整图片大小（建议宽度：12-14cm）

### 截图与占位符对应表

| Word 文档位置 | 截图文件 |
|--------------|----------|
| Step 1 占位符 | `step1_select_device.png` |
| Step 2 占位符 | `step2_guide_glucose.png` |
| Step 3 占位符 | `step3_searching.png` |
| Step 4 占位符 | `step4_device_list.png` |
| Step 5 占位符 | `step5_pairing.png` |
| Step 6 占位符 | `step6_success.png` |
