# 情商培养物料自动生成体系

## 目录结构

```
eq-materials/
├── library/          # 案例库
│   ├── literature/   # 文学作品案例
│   ├── movies/       # 影视作品案例
│   └── hot-topics/   # 热点案例
├── templates/        # 物料模板
├── scripts/          # 生成脚本
├── output/           # 生成输出
└── config/           # 配置文件
```

## 情商维度分类

1. **自我认知** - 了解自己的情绪、优势、局限
2. **情绪管理** - 调节情绪、压力管理
3. **自我激励** - 目标驱动、韧性
4. **同理心** - 理解他人感受、换位思考
5. **社交技能** - 沟通、冲突处理、影响力

## 使用方式

- 手动生成：运行 scripts/generate.sh
- 自动生成：cron 定时任务每日 8:00 执行
