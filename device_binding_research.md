## 一、三诺血糖仪 臻准 2000 (BA-2000)

### 产品特性
- **品牌**: 三诺 (Sinocare)
- **型号**: 臻准 2000 / BA-2000
- **连接方式**: 蓝牙 4.0 BLE
- **配套 APP**: 三诺健康 / 诺安健康

### 典型绑定流程
1. **设备准备**
   - 安装试纸或长按开机键 3 秒开机
   - 屏幕显示蓝牙图标（闪烁表示可配对）
   - 部分型号需要进入"设置"→"蓝牙"菜单

2. **手机操作**
   - 打开 APP → 设备管理 → 添加设备
   - 选择"血糖仪"→"臻准 2000"
   - APP 开始搜索蓝牙设备

3. **配对确认**
   - 手机显示设备名称（如：Sinocare_XXXX）
   - 部分型号需要确认配对码（6 位数字）
   - 设备屏幕显示"配对成功"或蓝牙图标常亮

4. **绑定完成**
   - APP 显示设备已连接
   - 可进行血糖测试和数据同步

### 注意事项
- ⚠️ 设备在测试状态下无法配对（需待机状态）
- ⚠️ 配对距离 ≤ 1 米
- ⚠️ 首次配对后自动重连

## 二、三诺血压计 BA-803

### 产品特性
- **品牌**: 三诺 (Sinocare)
- **型号**: BA-803
- **连接方式**: 蓝牙 4.0 BLE
- **供电**: 4 节 7 号电池 或 USB 供电
- **配套 APP**: 三诺健康

### 典型绑定流程
1. **设备准备**
   - 安装电池
   - 正确缠绕袖带
   - 长按电源键开机（约 3 秒）
   - 屏幕显示蓝牙图标

2. **进入配对模式**
   - 部分型号：开机后自动进入配对模式（蓝牙图标闪烁）
   - 部分型号：需要长按"记忆"键 5 秒进入配对
   - 屏幕显示"PC"或蓝牙符号

3. **手机操作**
   - 打开 APP → 设备管理 → 添加设备
   - 选择"血压计"→"BA-803"
   - APP 搜索并显示设备

4. **配对确认**
   - 选择设备（显示 MAC 地址后 4 位）
   - 部分型号需要确认配对码
   - 设备发出"滴"声或显示配对成功

5. **绑定完成**
   - APP 显示设备已连接
   - 可进行血压测量和数据同步

### 注意事项
- ⚠️ 测量状态下无法配对（需待机）
- ⚠️ 配对时袖带应处于放松状态
- ⚠️ 低电量可能影响蓝牙连接

## 三、蓝牙配对技术细节

### BLE 配对流程（标准）
```
1. 广播 (Advertising)
   └─> 设备发送广播包（包含设备名称、服务 UUID）

2. 扫描 (Scanning)
   └─> 手机扫描并发现设备

3. 连接 (Connection)
   └─> 手机发起连接请求

4. 配对 (Pairing)
   └─> 交换安全密钥（可选配对码确认）

5. 绑定 (Bonding)
   └─> 保存配对信息，实现自动重连
```

### 常见设备名称格式
- 血糖仪：`Sinocare_BA2000_XXXX` 或 `SN-XXXX`
- 血压计：`Sinocare_BA803_XXXX` 或 `BP-XXXX`
- XXXX = MAC 地址后 4 位

### 配对码
- 部分型号使用**固定配对码**: `000000` 或 `123456`
- 部分型号使用**动态配对码**: 设备屏幕随机显示 6 位数字
- 部分型号**无需配对码**: 直接确认连接

## 四、建议的 APP 绑定流程

### 页面流程
```
1. 选择设备类型 → 2. 操作指引 → 3. 搜索设备 → 
4. 选择设备 → 5. 配对确认 → 6. 绑定成功
```

### 关键交互点

| 步骤 | 用户操作 | 设备状态 | APP 反馈 |
|------|----------|----------|----------|
| 准备 | 设备开机 | 蓝牙图标闪烁 | 提示"设备已就绪" |
| 搜索 | 点击"开始搜索" | 广播模式 | 搜索动画 |
| 选择 | 点击设备列表 | 等待连接 | 高亮选中 |
| 配对 | 确认配对码 | 显示配对码 | 配对码对比 |
| 完成 | 点击"完成" | 蓝牙常亮 | 成功提示 |

### 异常处理
- **搜索超时**（30 秒无设备）→ 提示"未找到设备，请重试"
- **配对失败** → 提示"配对失败，请确认配对码"
- **连接断开** → 提示"连接断开，请检查设备电量"

## 五、当前 HTML 流程对比

### ✅ 已符合实际的功能
- [x] 6 步绑定流程设计
- [x] 操作指引页面
- [x] 搜索动画
- [x] 设备列表选择
- [x] 配对码确认
- [x] 成功状态

### ⚠️ 需要调整的功能
1. **配对码显示方式**
   - 当前：随机生成 6 位数字
   - 建议：支持"无需配对码"和"动态配对码"两种模式

2. **设备名称显示**
   - 当前：固定显示"Sinocare BA-2000"
   - 建议：显示实际广播名称（含 MAC 后 4 位）

3. **超时处理**
   - 当前：固定 3 秒后显示结果
   - 建议：添加"未找到设备"状态和重试按钮

4. **设备特定指引**
   - 当前：通用 4 步指引
   - 建议：根据设备型号显示特定操作（如血压计需要缠绕袖带）

## 七、建议更新

基于调研，建议对 HTML 原型进行以下更新：

1. **添加"无需配对码"选项** - 部分三诺设备直接连接
