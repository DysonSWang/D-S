# 任务申请与审批系统 + 里程碑 + 故事集 + 商城积分 + 小屋装扮

一个为**真人主奴关系**设计的综合管理与情感记录工具。

**核心功能**：
- ✅ 任务申请与审批系统
- 💕 在一起时间记录
- 🎯 里程碑节点管理
- 📖 双方故事集/日志（支持图文）
- 🏪 商城积分系统（虚拟装饰 + 精美礼物）
- 🏠 小屋装扮系统

## 🚀 使用方法

```bash
# ===== 任务命令 =====

# 奴 - 申请任务
python3 task_cli.py apply "学习 Python 基础" "完成入门教程" 7

# 奴 - 随机抽取任务（有积分奖励）
python3 task_cli.py random 7

# 奴 - 完成任务（获得积分）
python3 task_cli.py complete TASK-0001 "已完成"

# 主人 - 审批任务
python3 task_cli.py approve TASK-0001 yes "好好完成"


# ===== 商城命令 =====

# 浏览商城（按分类显示）
python3 task_cli.py shop

# 查看积分余额
python3 task_cli.py shop balance

# 充值积分（主人）
python3 task_cli.py shop recharge 500

# 兑换装饰物品
python3 task_cli.py shop redeem DECOR-001

# 兑换精美礼物
python3 task_cli.py shop redeem GIFT-003

# 查看兑换记录
python3 task_cli.py shop orders


# ===== 小屋命令 =====

# 查看我的小屋
python3 task_cli.py room

# 查看拥有的物品
python3 task_cli.py room items

# 装饰小屋
python3 task_cli.py room decorate


# ===== 里程碑命令 =====

python3 task_cli.py milestone init 2024-01-15
python3 task_cli.py milestone show
python3 task_cli.py milestone add "第一次旅行" "去了云南"
python3 task_cli.py milestone list


# ===== 故事集命令 =====

python3 task_cli.py story add "第一次约会" "今天一起去了公园"
python3 task_cli.py story add "生日惊喜" "主人准备了礼物" ./photo.jpg
python3 task_cli.py story list
python3 task_cli.py story view STORY-0001
python3 task_cli.py story search 晚餐
python3 task_cli.py story tag STORY-0001 温馨
python3 task_cli.py story timeline
```

## 🎯 任务状态流转

```
                    待审批
                   /     \
           批准   /       \  拒绝
                 /         \
             已批准        已拒绝
             /    \
        完成/      \ 延期 (限 1 次)
           /        \
       已完成      已延期
       (得积分)      \
                      \ 超期
                       \
                       已失败
```

## ⚠️ 注意事项

1. **数据备份**：定期备份 `~/.task_manager/` 目录
2. **隐私保护**：数据存储在本地，不会上传
3. **积分规则**：
   - 完成任务才能获得积分
   - 超期失败任务无积分
   - 积分可以充值（主人）
4. **小屋装饰**：
   - 虚拟装饰物品兑换后自动添加到小屋
   - 实物礼物不会添加到小屋
   - 装饰功能开发中
